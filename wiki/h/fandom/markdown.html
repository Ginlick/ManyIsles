Many Isles Markdown

Markdown is a common way of writing content and representing different types of text, such as headers, normal paragraphs, or bold text. In the fandom wiki, we use a special version called Many Isles Markdown. If you know html, you can also use that.
To test any Many Isles markdown, feel free to use our Markdown Editor!

Markdown Syntax

Basics

#header You can write headers using hashtags. A single one shouldn't be used as it designates a title, while ## gives header 2 and ### header 3. Notice that in side-thumbnails, header 4s have a special look.
*italics* A single pair of asterisks displays text in italics.
**bold** A double pair of asterisks displays text in bold.
***bold italics*** A triple pair of asterisks displays text in bold and italics.
[link](url) You can embed links like this, with the square brackets being the displayed text, while the standard ones are the actual url.

Thumbnails

To display all the children of an article, you can write [wiki:children] in your text; this will be replaced by a nice group of thumbnails. We recommend using this on "category" articles, but not on "essay" articles. Note: you can only do this once in an article.
Use the shortcut ctrl + shift + l to create thumbnails for single articles.

Images

We have some basic code for images.

<div class="sideimg"><img src="directurl.jpg" /><p>caption.</div>
<div class="sideimg landscape"><img src="directurl.jpg" /><p>caption.</div>

Optionally, you can use actual html markup. The image to the right is displayed using this code:

<img src='https://manyisles.ch/IndexImgs/Admin.jpg' alt='oops' style='display:inline-block;width:20%;padding:10px;float:right;max-width:190px;border-radius:22px;' />

Here are some further examples of image formatting:

<img src='https://manyisles.ch/Imgs/Ranks/Poet.png' alt='oops' style='display:block;width:80%;margin:auto;' />
<img src='/Imgs/SunMagic.png' alt='oops' style='display:inline-block;width:20%;padding:10px;float:right;max-width:200px' />

To embed an image into our markdown, it needs to be hosted on the internet and directly accessible, as for other editors such as the homebrewery. You can alternatively directly embed one image in the side-thumbnail.

Popup Text Content

You can make popup blocks using the code below:

<h3 onclick='showBg(1);'>Header Text <i class='fas fa-angle-down arrow' id='f1'></i></h3>
<div class='hidhidCont' id='h1'>
<div id='1' class='hiddenCont'>
<p>
Hidden Body
</p>
</div>
</div>

Make sure the call number (in this case, 1) is unique in the whole article.
